<p align="center">
</p>

在使用ipygo进行二次开发（策略、模块等）的过程中有任何疑问，请查看[**ipygo项目文档**]

## 功能特点

1.针对具体策略算法和功能开发提供了简洁易用的模板，用于快速构建交易员所需的量化交易应用。

## 环境准备

* 支持的系统版本：Windows 10以上/Windows Server 2016以上/Ubuntu 20.04 LTS以上
* 支持的Python版本：Python 3.8 64位（**注意必须是Python 3.7 64位版本**）
